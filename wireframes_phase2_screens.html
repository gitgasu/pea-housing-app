Design Decisions - Search & Filter

"No broker fee" default selected: First chip, checked by default
Research: Mr. Patterson's #1 priority - "avoid large broker fees" - make this the easiest filter to apply
"Accepts vouchers" prominent: Second filter option
Research: Ms. Ellis's critical need - voucher-accepting landlords. Both interviewees mentioned landlord discrimination.
Accessibility filter visible: Wheelchair accessible option
Research: 100% rated accessibility as at least somewhat important; 50% rated as essential
Filter chips + detailed panel: Quick filters on top, detailed below
Plain language: "Important to Me" not "Amenities" or "Requirements"
Result count shown: "Show 24 Results" = transparency, sets expectations
Reset option: Easy to start over (Mr. L needs simplicity)
Search by neighborhood: Users may not know exact addresses
